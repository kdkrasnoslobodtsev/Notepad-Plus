Здравствуй, уважаемый проверяющий! Добро пожаловать в приложение "Notepad+" - текстовый редактор, авторского производства.
В нем ты можешь найти следующие функции:
1. Открывать, редактировать и сохранять простые текстовые файлы (без
сохранения форматирования).

2. Открывать, редактировать и сохранять текстовые файлы формата *.rtf (с
сохранением форматирования).

3. Предоставить возможность открывать несколько текстовых файлов в
одном окне приложения с возможностью переключения между ними (с
несколькими вкладками).

Каждая вкладка имеет имя вида "tabPage{номер вкладки"}
Чтобы открыть новую вкладку, следует открыть меню "Файл", нажать "Создать"
!!!Открытие существующего файла осуществляется в открытую на данный момент вкладку.

4. При наличии несохраненных изменений в открытых файлах при
закрытии приложения предоставить пользователю возможность
сохранить файл, не сохранять или отменить закрытие приложения.

5. Реализовать меню приложения со следующими подменю: Файл, Правка,
Формат(для выбора основного шрифта текста), Настройки. Список
может быть дополнен на ваше усмотрение.
Файл      - Открыть
          - Сохранить как
          - Сохранить
          - Создать
          - Выход
Правка    - Копировать
          - Вставить
          - Вырезать
          - Выделить все

Формат 	  - Настройки Шрифта

Настройки - Настройки фона
	  - Настройки сохранения - Частота сохранения

В мею "Формат" нажать на кнопку "Настройки шрифта" - появится окно для выбора шрифтов, размеров, стили.
Если при этом не выбран фрагмент текста, то изменение применяется ко всему документу, иначе - только к ввыбранному фрагменту.

6. Реализовать автоматическое сохранение открытых файлов. Частота
сохранения задаётся в окне Настроек.
Автосохранение начинается после первого сохранения файла. Период между сохранениями указан в секундах(по умолчанию стоит 30).
Чтобы изменить интервал времени - открыть меню "Настройки", "Настройки сохранения", "Частота сохранения", записать в текстовое окно нужный интервал. Некорректные входные данные будут обработаны.

7. Реализовать возможность выбора цветовой схемы приложения в
настройках.
Чтобы изменить цвет приложения - открыть меню "Настройки", "Настройки фона" - появится окно для выбора цвета.

8. Предоставить возможность пользователю форматировать выделенный
фрагмент изменяя стиль текста по следующим параметрам:
• Курсив;
• Жирный;
• Подчёркнутый;
• Зачёркнутый.
Причём, несколько стилей могут применяться к выделенному фрагменту
одновременно.
Инструкция указана в пункте 5

9. Создать контекстное меню, активируемое правой кнопкой мыши, со
следующим функционалом:
• Выбрать весь текст;
• Вырезать выделенный фрагмент;
• Копировать выделенный фрагмент;
• Вставить сохранённый в буфере обмена фрагмент;
• Задать формат выделенного фрагмента текста
